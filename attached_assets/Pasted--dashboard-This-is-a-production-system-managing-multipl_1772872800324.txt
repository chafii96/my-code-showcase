## CORE SYSTEM: Multi-Provider API Manager

### Providers to integrate:

1. Ship24 (primary)

2. TrackingMore (secondary)

3. 17Track (tertiary)

4. Custom Scraper (fallback)

### Each provider has:

- Multiple API accounts (Account 1, 2, 3, 4, 5...)

- Auto-rotation between accounts when quota is exhausted

- Enable/disable toggle per account

- Priority order (drag to reorder)

- Daily quota limit tracking

- Used requests counter

- Success rate percentage

- Average response time (ms)

- Last used timestamp

- Status indicator (active/exhausted/error/disabled)

### Failover Logic:

1. Try Account 1 of Provider 1 (Ship24)

2. If quota exhausted → switch to Account 2 of Ship24

3. If all Ship24 accounts exhausted → switch to Provider 2 (TrackingMore)

4. If TrackingMore exhausted → switch to 17Track

5. If 17Track exhausted → use Custom Scraper

6. If all fail → return cached result or error

7. Reset quotas daily at midnight UTC

## SMART CACHE SYSTEM (Advanced 1-hour base)

### Logic:

- Base TTL: 1 hour

- Smart TTL based on status:

  * "Delivered" → cache 24 hours (won't change)

  * "Out for Delivery" → cache 20 minutes (changes fast)

  * "In Transit" → cache 2 hours

  * "Pending" → cache 1 hour

  * "Exception" → cache 15 minutes

  * "Pre-shipment" → cache 1 hour

- Negative caching: cache "not found" results for 30 mins to avoid repeated API calls

- Cache warming: if a tracking number is searched 3+ times, 

  automatically refresh it every 30 minutes proactively

- Stale-while-revalidate: return cached data immediately, 

  refresh in background if TTL is 80% expired

## AUTOMATIC FAILOVER ENGINE

```javascript

// Pseudocode for failover logic

async function trackPackage(trackingNumber) {

  

  // Step 1: Check cache

  const cached = await cache.get(trackingNumber);

  if (cached && !isStale(cached)) return cached;

  

  // Step 2: Get ordered list of active accounts

  const accounts = await getActiveAccountsSorted(); 

  // Returns: [Ship24-Acc1, Ship24-Acc2, Ship24-Acc3, 

  //           TrackingMore-Acc1, 17Track-Acc1, Scraper]

  

  // Step 3: Try each account in order

  for (const account of accounts) {

    if (account.quotaExhausted) continue;

    if (!account.enabled) continue;

    

    try {

      const result = await callAPI(account, trackingNumber);

      await updateAccountStats(account, 'success');

      await cache.set(trackingNumber, result);

      return result;

    } catch (err) {

      await updateAccountStats(account, 'error');

      if (err.code === 'QUOTA_EXCEEDED') {

        await markAccountExhausted(account);

      }

      continue; // try next account

    }

  }

  

  // Step 4: All failed, return stale cache if exists

  if (cached) return { ...cached, stale: true };

  

  throw new Error('All providers exhausted');

}
```
