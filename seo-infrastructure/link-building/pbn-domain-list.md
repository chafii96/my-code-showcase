# PBN Domain List & Acquisition Guide 2026
# For: uspostaltracking.com

## What is a PBN?
A Private Blog Network (PBN) is a network of expired domains with existing authority that you control. 
Each domain hosts a website that links back to your money site (uspostaltracking.com).

## PBN Setup Instructions

### Step 1: Register Domains Anonymously
- Use different registrars for each domain (GoDaddy, Namecheap, Google Domains, etc.)
- Use privacy protection on all domains
- Pay with different payment methods if possible
- Use different email addresses for each registrar

### Step 2: Hosting Setup
- Use different hosting providers for each PBN domain
- **Never host multiple PBN sites on the same IP**
- Recommended providers:
  - Cloudflare Pages (free, different IPs)
  - GitHub Pages (free, different IPs)
  - Netlify (free, different IPs)
  - Different shared hosting accounts

### Step 3: Content Creation
Each PBN site needs:
- 5-10 unique articles (use AI + humanizer)
- About page, Contact page, Privacy Policy
- Relevant to the domain's original niche
- 1-2 links to uspostaltracking.com per site (not more)

### Step 4: Footprint Avoidance
- Different WordPress themes per site
- Different plugins per site
- Different Google Analytics IDs (or no GA)
- Different contact emails
- Different whois info
- Never interlink PBN sites with each other
- Never link from PBN to PBN

## Risk Management

### Signs Your PBN Was Detected:
- Manual action in Google Search Console
- Sudden drop in rankings
- Links no longer showing in Ahrefs

### What to Do If Detected:
1. Remove the links from the penalized PBN site
2. Disavow the domain in Google Search Console
3. Do NOT panic — one penalized PBN site won't kill your main site
4. Replace with a new PBN domain

### Footprint Checklist:
- [ ] Different registrars
- [ ] Different hosting IPs
- [ ] Different themes
- [ ] Different content styles
- [ ] No interlinking between PBN sites
- [ ] No shared analytics
- [ ] No shared contact info
- [ ] Privacy protection on all domains
